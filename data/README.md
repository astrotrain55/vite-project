#### Команды:
```bash
# Создать компоненты:
npm run create компонент1 компонент2 ... компонентN
```

#### Поиск:
```bash
# Искать списком:
npm run list строка

# Искать подробно:
npm run once строка

# Искать всё:
npm run full строка
```
